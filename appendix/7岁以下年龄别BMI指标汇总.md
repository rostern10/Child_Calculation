# 7岁以下年龄别BMI指标汇总

## 指标名称

- 年龄别 BMI
- 输入项：性别、年龄（月龄/年龄）、体重（kg）、身长或身高（cm）、测量方式
- 作用：判断消瘦、重度消瘦、超重、肥胖、重度肥胖

## 计算公式

`BMI = 体重(kg) / 身高(m)^2`

- 特殊规则：
- 24月龄存在 `2a`、`2b` 两行
- `2a`：24月龄按身长口径
- `2b`：24月龄按身高口径
- 文件注释说明：
- 0-2岁 BMI 按身长计算
- 如果 0-2岁测量的是立位身高，需要在身高基础上 `+0.7cm`，换算成身长后再算 BMI
- 如果 2-3岁测量的是卧位身长，需要在身长基础上 `-0.7cm`，换算成身高后再算 BMI

## 判定含义

年龄别 BMI 的判定与图一完全一致：

- `>= +3SD`：重度肥胖
- `+2SD <= 数值 < +3SD`：肥胖
- `+1SD <= 数值 < +2SD`：超重
- `-1SD <= 数值 < +1SD`：--
- `-2SD <= 数值 < -1SD`：--
- `-3SD <= 数值 < -2SD`：消瘦
- `< -3SD`：重度消瘦

## 系统实现建议

- BMI 不是直接录入的标准表字段，而是先计算再查表
- 2岁附近要先做身长/身高换算，再计算 BMI
- 最终输出字段建议：
- 指标名称：年龄别 BMI
- BMI 实际值
- SD区间
- 判定结果
- 依据来源
