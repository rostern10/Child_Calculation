# 7岁以下身长身高别体重指标汇总

## 指标名称

- 身长/身高别体重
- 输入项：性别、身长或身高（cm）、体重（kg）、测量方式
- 作用：判断消瘦、重度消瘦、超重、肥胖、重度肥胖

## 判定含义

这个指标是图一里最核心的营养状态指标之一：

- `>= +3SD`：重度肥胖
- `+2SD <= 数值 < +3SD`：肥胖
- `+1SD <= 数值 < +2SD`：超重
- `-1SD <= 数值 < +1SD`：--
- `-2SD <= 数值 < -1SD`：--
- `-3SD <= 数值 < -2SD`：消瘦
- `< -3SD`：重度消瘦

## 系统实现建议

- 这个指标不依赖年龄，核心是“当前身长/身高对应的体重区间”
- 但系统仍需结合年龄判断该走身长表还是身高表
- 2岁附近一定要先确定测量方式再查表
- 最终输出字段建议：
- 指标名称：身长/身高别体重
- SD区间
- 判定结果
- 依据来源
